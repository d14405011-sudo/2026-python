## 資源優化建議

由於 VT 資料夾中的圖片較大（23MB+），若要直接集成，建議：
1. 使用圖片編輯工具縮小至 1920x1080 或更小
2. 轉換為網頁優化的格式（WebP）
3. 創建適應不同遊戲解析度的圖片變體
4. 實施懶加載機制
