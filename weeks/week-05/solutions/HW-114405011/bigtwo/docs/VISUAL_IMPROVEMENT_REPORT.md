### 技術細節

**主題配置結構：**
```python
THEMES = {
    'theme_name': {
        'name': '顯示名稱',
        'description': '描述',
        'bg_top': (R, G, B),           # 背景頂部漸層
        'bg_bottom': (R, G, B),        # 背景底部漸層
        'felt_line': (R, G, B),        # 牌桌線條
        'glow_color': (R, G, B),       # 光斑顏色
        'glow_alpha': 32,              # 光斑透明度
        'card_back': (R, G, B),        # 卡牌背景主色
        'card_back_dark': (R, G, B),   # 卡牌背景暗色
        'button_primary': (R, G, B),   # 按鈕顏色
        'button_hover': (R, G, B),     # 按鈕懸停色
    }
}
```

**光斑動畫公式：**
- X軸：sin(時間 + 偏移) × 40
- Y軸：cos(時間 × 0.7 + 偏移) × 30
- 6層漸進式光暈（由外到內透明度遞增）
