# 牌遊戲美術強化計畫

## 參考風格分析
- **冬季風格** (`黑月星塵 冬.png`)：深藍色夜空 + 暖金色燈光
- **霓光風格** (`01.霓光 Neon Light.png`)：賽博朋克彩虹霓光
- **夏日風格** (`02.戀夏 Summer.png`)：明亮清爽色調
- **KSP Logo 變體**：可作為APP LOGO參考

## 實施方案

### 第一階段：顏色精細化（立即實施）
1. **牌桌背景優化**
   - 冬季主題：深藍漸層 + 金色光斑
   - 霓光主題：深紫 + 青藍 + 粉紅霓光
   - 淺色主題：淺灰 + 綠絨背景

2. **卡牌外觀升級**
   - 加強卡牌邊框陰影
   - 改進花色顏色對比（黑/紅更鮮明）
   - 添加亮度高亮在卡牌角落

3. **UI按鈕美化**
   - 漸層按鈕背景
   - 懸停動畫增強
   - 模式選擇按鈕視覺區分

### 第二階段：裝飾元素（可選）
1. **角落裝飾**
   - 四角淡光效果（參考KSP風格）
   - 動態粒子效果（牌局開始時）

2. **模式指示器**
   - 休閒模式：明亮色調 + 笑臉圖標
   - 挑戰模式：暗色調 + 火焰圖標

3. **勝利畫面**
   - 應勝者頭像區域（預留位置）
   - 五彩紙屑動畫

### 第三階段：進階集成（未來）
- 資源優化後可集成VT資料夾中的角色圖片
- 製作專用的卡牌背設計圖
- APP LOGO設計

## 優先級
🔴 **高**：牌桌背景、卡牌視覺、UI漸層  
🟡 **中**：角落裝飾、動畫效果  
🟢 **низ**：角色圖片、專用資源製作
