## 🎯 功能特性

### 主題系統 API
```python
# 初始化時指定主題
renderer = Renderer(width, height, theme='winter')

# 運行時切換主題
renderer.set_theme('summer')

# 取得所有可用主題
from ui.themes import get_all_themes
themes = get_all_themes()  # ['default', 'winter', 'neon', 'summer']
```

### 自動主題應用（在遊戲開始時）
```python
# 挑戰賽模式 → 冬季夜晚
game = BigTwoGame(difficulty='nightmare', mode='ranked')
# 自動應用 'winter' 主題

# 休閒模式 → 夏日明朗
game = BigTwoGame(difficulty='easy', mode='casual')
# 自動應用 'summer' 主題
```

## 🎨 設計參考

### VT 資料夾參考
- 冬季主題靈感：`黑月星塵 冬.png`（深藍夜空 + 金色光）
- 夏日主題靈感：`02.戀夏 Summer.png`（明亮清爽）
- 霓光主題靈感：`01.霓光 Neon Light.png`（賽博朋克）
- 角色設計參考：6 位角色 × 12-25 張月份變體 = 80+ 圖片

### 色彩系統
所有顏色定義在 `assets/colors.json`，支持 RGB 和 HEX 格式：
```json
{
  "winter": {
    "bg_top": "#123340",      // RGB (18, 51, 64)
    "bg_bottom": "#081220",   // RGB (8, 18, 32)
    ...
  }
}
```
